# [사이트 방문하기](http://recipesoup.koreacentral.cloudapp.azure.com/)

## 개요
- 영양사들이 자신의 식단을 공유할 수 있는 커뮤니티 입니다.
- 추후에 추가될 서비스 목표는 검증된 영양사 사용자들로부터 일반 사용자가 식단을 제공 받는 것입니다.
